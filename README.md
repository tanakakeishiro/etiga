# Web Coding Template

Web制作（コーディング）で使用する **自分用テンプレートリポジトリ** です。

HTML / SCSS / JavaScript を用いた静的サイト制作を想定し、
**保守性・再利用性・実務運用** を意識した構成になっています。

---

## 概要

* 静的Webサイト制作向けテンプレート
* 新規案件ごとに **GitHub Template Repository** として複製して使用
* 初期構成・設定ファイルを共通化し、制作スピードを向上させる目的

---

## 想定用途

* Web制作の学習用
* クライアント案件（静的サイト）
* ポートフォリオ制作

---

## 使用方法

### 1. テンプレートから新規リポジトリ作成

GitHub 上で **Use this template** をクリックし、案件用リポジトリを作成してください。

---

### 2. パッケージインストール

```bash
npm install gulp -D
```

---

### 3. 開発開始

```bash
npm run dev
```

※ 実行コマンドは `package.json` の scripts に準拠してください。

---

## コーディング方針

* クラス設計：BEM記法
* SCSS構成：役割ごとに分割管理
* HTML：セマンティックHTMLを意識
* 保守性・可読性を優先

---

## Git運用ルール

### 管理対象

* HTML / SCSS / JS
* 設定ファイル（gulp / lint / formatter）
* README

### 管理しないもの

* `node_modules/`
* `dist/`（ビルド成果物）
* `.env` 等の環境依存ファイル

---

## 注意事項

* 本テンプレートは **案件用の雛形** です
* このリポジトリを直接編集して案件を進めないでください
* 必ずテンプレートから複製して使用してください

---

## ライセンス

このテンプレートは個人利用を前提としています。
必要に応じて各案件の要件に合わせて調整してください。
